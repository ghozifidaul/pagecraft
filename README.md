# PageCraft API

PageCraft is a web application designed to help parents generate personalized children's books with stories and illustrations generated by AI.

## Tech Stack

- **Back-End:** [Hono](https://hono.dev/) (running on Cloudflare Workers)
- **Front-End:** [Vite](https://vitejs.dev/)
- **Database:** [Cloudflare D1](https://developers.cloudflare.com/d1/)
- **Storage:** [Cloudflare R2](https://developers.cloudflare.com/r2/)

## Prerequisites

- [Node.js](https://nodejs.org/) (latest LTS)
- [Wrangler CLI](https://developers.cloudflare.com/workers/wrangler/) (`npm install -g wrangler`)
- Gemini API Key (set in environment)

## Setup

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Configure Environment:**
   Create a `.env` file (based on `.env.example`) and configure the necessary secrets (Gemini API Key, R2 credentials).

3. **Development:**
   ```bash
   npm run dev
   ```

4. **Deploy:**
   ```bash
   npm run deploy
   ```

## Scripts

- `npm run dev` — Start development server
- `npm run deploy` — Deploy to production
- `npm run cf-typegen` — Synchronize Cloudflare Worker types
- `npm run test` — Run unit tests
- `npm run test:all` — Run all test suites
